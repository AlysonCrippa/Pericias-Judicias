Configuração da API
Por que precisa de proxy? O GitHub Pages bloqueia chamadas diretas à API da Anthropic (política CORS do navegador). O Cloudflare Worker é um intermediário gratuito que resolve isso.

Como criar o proxy (5 minutos, gratuito):
1. Acesse workers.cloudflare.com e crie uma conta gratuita
2. Clique em Create Worker
3. Substitua TODO o código pelo código abaixo e clique em Deploy
4. Copie a URL do worker (ex: https://pericia-proxy.seu-usuario.workers.dev) e cole acima

Código do Worker (copie e cole inteiro):
export default { async fetch(request) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST, OPTIONS', 'Access-Control-Allow-Headers': '*', }}); } const url = new URL(request.url); const target = 'https://api.anthropic.com' + url.pathname; const res = await fetch(target, { method: request.method, headers: request.headers, body: request.body, }); const headers = new Headers(res.headers); headers.set('Access-Control-Allow-Origin', '*'); return new Response(res.body, { status: res.status, headers, }); } };
